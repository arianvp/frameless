Frameless

A CSS framework that does it right

Grid System

The grid system of Frameless uses flexbox and is implemented as two Sass mixins.

Using Sass mixins instead of CSS classes allows for greater separation of concerns; no ugly HTML littered with grid-related CSS classes. 

<div id='container'>
    <section id='content'>
        <p>Lorem ipsum.</p>
    </section>
    <section id='sidebar'>
        <p>Hi, I'm awesome.</p>
    </section>
</div>
@import "grid.sass"

$col-gap: 20px

@mixin col($width)
    @include fl-col($width, $col-gap)

#container
    @include fl-row()

    & > #content
        @include col(8)

    & > #sidebar
        @include col(4)
